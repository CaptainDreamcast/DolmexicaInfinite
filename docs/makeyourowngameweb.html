Make-your-own-game for Web documentation

Toolkit to make your own Dolmexica Infinite game for HTML5. It is a downloadable toolkit separate from base Dolmexica Infinite. It contains a .bat file that is used as a command line tool as well as a program that serves as a graphical interface for it.

GUI

Open "Dolmexica Infinite Make-your-own-game kit for Web.exe"

Batch script command

make_web.bat GameName

Example

make_web.bat DolmexicaInfiniteDemo100

How to use

To mod the game, change the files in the assets folder just like you would with Mugen.
If you want to modify the index.html used to display the game, for example to display your own logo, modify common/index.html.
The batch file will output a zipped folder you can directly upload to itch.io, newgrounds, or similar sites.

Known issues

  • Requires PowerShell 3 for zipping so that step might not work with older Windows versions. The contents of the zip file are also in the build folder, so if automatic zipping fails, zip that one manually.
  • Opening the index.html in the build folder in your local browser will not work. To test the game, upload it to one of the sites above, or use emrun from emscripten.

Third-party tools used in the toolkit

  • Extremely slimmed down version of portable python (https://sourceforge.net/projects/winpython/)
  • Extremely slimmed down version of emscripten (https://emscripten.org/)
